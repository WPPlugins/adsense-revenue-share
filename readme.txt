AdSense Revenue Share plugin allows you to share AdSense revenue with authors.

== Description ==

AdSense Revenue Share plugin developed by: [CozyWP](http://cozywp.com)

This plugin allows to share your AdSense revenue with blog post author. After filling the Publisher ID field in account settings, an author receives a percentage of advertisement impressions. Percents amount can be customized in plugin settings, by default itâ€™s 50%.

= Features =

* Revenue share for admin and authors
* Simple one-page settings
* Customizable revenue percentage for authors
* Customizable banner size (only horizontal banners)
* Customizable banner position, top, bottom or body of content
* Advertisement widget
* Publisher ID validation. Admin advertisement is shown if author sets incorrect Publisher ID

Official page: [http://cozywp.com/adsense-revenue-share/](http://cozywp.com/adsense-revenue-share/)


== Installation ==

1. Upload plugin to the '/wp-content/plugins/' directory.
2. Activate plugin through 'Plugins' menu in WordPress Admin Panel.
3. Go to Settings - Adsense Rev Share and customize plugin as you need.
